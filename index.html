Interactive Algorithms Lab

Simulators and Calculators.

1) Bubble Sort Simulator

Enter comma-separated numbers and watch the passes that gradually sort the list.

2) Binary Search Calculator

Search for a target inside a sorted list and inspect the midpoints visited.

3) Fibonacci Calculator

Compute Fibonacci using dynamic programming so learners see the table being built.

4) KMP String Matcher

Search for a pattern in text and inspect the LPS table used by Knuth-Morris-Pratt.

5) Linear Regression Calculator

Enter X and Y values and get the slope, intercept, and predictions using the simple formula.

6) Logistic Regression Probability Demo

This calculator uses sigmoid(b + w1x1 + w2x2) so learners can understand probabilities and classification.

7) Dijkstra Shortest Path

Use a small weighted graph and compute shortest distances from a starting node.

8) Minimum Spanning Tree: Kruskal / Prim

Experiment with the same graph using two classic minimum spanning tree algorithms.

9) Neural Network Playground

A tiny two-input, one-hidden-layer neural network demo. Adjust inputs, weights, and biases to see forward propagation step by step.

10) Stack Simulator

Learn the stack data structure interactively using push and pop operations.

11) Queue Simulator

Practice queue operations with enqueue and dequeue.

12) Linked List Builder

Add nodes to a singly linked list and remove the head node.

13) Binary Search Tree Playground

Insert values into a binary search tree and view traversals.

14) Hash Table Demo

Store key-value pairs in a simple hash table and inspect the bucket where each item lands.

15) TOY Computer Simulator

A tiny virtual machine simulator with 16 registers, 256 memory cells, a program counter, fetch-decode-execute logic, and the rule that R0 always stays zero.
